# Comment faire fonctionner ce code

Faire fonctionner le starter pour réaliser le DM fait parti de l'exercice.

Si vous avez des erreurs lors du lancement via laragon, essayez de les corriger, cela sera pris en compte dans la note.

## Sans laragon
Installez PHP, [ici](https://www.tutorials24x7.com/php/how-to-install-php-8-on-windows) un guide pour le faire sur windows. 
Une fois l'installation correctement effectué. la commande `php` doit être disponible dans votre terminal. 

Dans ce cas clonez le repo et executez la commmande `php -S localhost:8000` à la racine du repository. 

Rendez vous sur http://localhost:8000 sur un navigateur pour voire le projet.
